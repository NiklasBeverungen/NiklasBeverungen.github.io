Example: title option

You can put html code in the attribute.

HTML:

<div id="container">
    <ul>
        <li>
            <a href="#">
            <img src="img/1.jpg" title="TV Series Character" alt="image 1">
            </a>
        </li>
        ...
        <li>
            <a href="#">
            <img src="img/4.jpg" title="<h4 style='text-align:left;margin:0px;padding:0px;'>Elly Tran Ha</h4><hr><p style='color:#yellow;text-align:left;'>A Vietnam model. Hot for her selfies</p>" alt="image 4">
            </a>
        </li>
        ...
    </ul>
</div>

JavaScript:

$('#container').sliphover({
    title: 'alt' //or 'title' or any other attribute
});
